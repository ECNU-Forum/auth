A [Flarum](http://flarum.org) extension. 

## Installation

Install with composer:

```sh
composer require ecnu-im/auth:"*"
```

## Updating

```sh
composer update ecnu-im/auth:"*"
php flarum migrate
php flarum cache:clear
```
